# 🌍 CodeAlpha Language Translator

An AI-powered Language Translation Tool built using **Python** and **Streamlit** as part of the **CodeAlpha Python Programming Internship**.

## 📌 Features

- Translate text between multiple languages.
- Simple and interactive web interface.
- Supports a wide range of languages.
- Fast and easy to use.
- Real-time translation.

## 🛠️ Technologies Used

- Python
- Streamlit
- Googletrans (or the translation library you used)

## 📂 Project Structure

```
language-translator/
│
├── Language Translation Tool.py
├── requirements.txt
└── README.md
```

## 🚀 Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/rechinthalavinay-cloud/CodeAlpha_Language_Translator.git
```

### 2. Navigate to the project folder

```bash
cd CodeAlpha_Language_Translator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run "Language Translation Tool.py"
```

### 5. Open in your browser

Visit:

```
http://localhost:8501
```

## 🌐 Features Demonstration

- Select the source language.
- Select the target language.
- Enter the text to translate.
- Click the **Translate** button.
- View the translated output instantly.
